Kodgranskning – ha_reader.py
============================
Datum: 2026-04-22


1. BUGG — `options` utanför scope (rad 57)
------------------------------------------
QDRANT_HOST och QDRANT_PORT refererar till `options` via en ternär operator,
men `options` är bara definierad inuti det föregående if-blocket.
Raden evalueras felaktigt och kraschar om OPTIONS_FILE inte finns.

Åtgärd: Läs QDRANT_HOST/PORT inuti respektive gren av if/else-blocket ovan,
eller definiera `options = {}` som fallback utanför blocket.


2. BUGG — System-prompt saknas i andra AI-anropet (rad 439)
------------------------------------------------------------
När modellen gör ett tool-anrop skapas ett `second_response` utan system-prompt.
Modellen tappar kontexten om vilka enheter som finns och vilka instruktioner som gäller.

Åtgärd: Lägg till system-meddelandet (med device_info och memory_keys)
även i messages-listan för second_response.


3. SÄKERHET — XSS i frontend (rad 683)
---------------------------------------
AI:ns svar injiceras direkt som innerHTML:
    row.innerHTML = '<div class="bubble ' + role + '">' + text + '</div>...'

Om modellen returnerar HTML-taggar (t.ex. <script>) körs de i browsern.

Åtgärd: Sätt bubblan med textContent istället, eller escapa strängen:
    const bubble = document.createElement('div');
    bubble.className = 'bubble ' + role;
    bubble.textContent = text;


4. PYTHON ANTI-PATTERN — Mutable default argument (rad 380)
------------------------------------------------------------
def ask_ai(user_message, user_history=[], session_id="okänd"):

Listan [] delas mellan alla anrop som inte skickar ett eget värde,
vilket kan ge oväntade sidoeffekter.

Åtgärd:
    def ask_ai(user_message, user_history=None, session_id="okänd"):
        user_history = user_history or []


5. OBEGRÄNSAD SESSION-HISTORIK
-------------------------------
sessions[session_id] växer utan begränsning. Långa konversationer
kan överskrida modellens token-gräns och ge fel eller höga kostnader.

Åtgärd: Begränsa historiken, t.ex. behåll bara de 20 senaste turerna:
    sessions[session_id] = sessions[session_id][-20:]
